1.create ec2 instance and connect 
2. install node.js
     sudo yum update -y
     curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
     sudo yum install nodejs -y
3.check version
     node -v
     npm -v
4.create dirctory and backend code deploy using nano
5. Install backend dependices
    npm install
    node server.js
6.contious running install pm
    npm install -g pm2
    pm2 start server.js
    pm2 save
    
7.check our backend 
    http://EC2_PUBLIC_IP:3000/api/message
8.paste backend url to frontend server.js 
9.paste all front code to using nano
10. copy and paste our index.html file to nginx
    sudo rm -rf /usr/share/nginx/html/*
    sudo cp -r * /usr/share/nginx/html/
    sudo systemctl restart nginx
